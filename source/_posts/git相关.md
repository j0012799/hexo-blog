* [gitlab/github多账户管理ssh](http://www.cnblogs.com/fanyong/p/3962455.html)

    > ssh 简单来说就是一种加密通信手段，服务器保存一段密文，你本地保存一段密文，互相通信就不再需要帐号密码

    > 所以，在 git 中提交代码时，如果不想每次输入帐号密码，可以用 ssh 达到验证账户的目的
